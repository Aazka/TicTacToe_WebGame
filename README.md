# ❌⭕ Tic Tac Toe

A simple and fun **Tic Tac Toe** game built with **HTML, CSS, and JavaScript**.

## 🎮 Features
- Two-player+AI gameplay
- Win & draw detection
- Reset game functionality
- Responsive UI

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript

## 🚀 Run Locally
Simply open `index.html` in your browser.

## 📚 Purpose
Built for practice to improve JavaScript, DOM manipulation, and front-end development skills.
